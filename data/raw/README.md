# Raw Data

This folder is intended to store the original raw datasets used in this project.

Due to size and access restrictions, the raw data files are not included in this repository.

The data used in this project comes from Environment Agency macroinvertebrate datasets, including:

- Site-level data
- Sampling metrics
- Taxa-level abundance data
- WHPT family-level metrics

These datasets were processed using a structured data cleaning pipeline before analysis.

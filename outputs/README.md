# Project Outputs

This folder contains the final outputs generated from the Freshwater Macroinvertebrates Analysis project.

## 📄 Report

* **report.pdf**
  A complete project report produced from the R Markdown workflow.

  The report includes:

  * Project motivation and environmental context
  * Data description and challenges
  * Data cleaning and preprocessing pipeline
  * Statistical modelling approaches (GAM, ordinal, censored models)
  * Exploratory data analysis and visualisations
  * Key findings on long-term biodiversity trends

## 🔍 Summary

The report presents a full end-to-end data science workflow, transforming large-scale, messy ecological data into structured insights. It demonstrates how different modelling approaches can be used to analyse environmental trends while accounting for uncertainty and data limitations.

## ♻️ Reproducibility

All outputs in this folder can be reproduced by running the scripts provided in the `scripts/` directory, starting from raw data ingestion through to final modelling.

## 📝 Notes

* The report is intended as a polished summary of the project and is suitable for academic and professional review.
* Due to data size constraints, only representative outputs are included in this repository.
